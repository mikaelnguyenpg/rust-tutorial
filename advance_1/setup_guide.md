1. Install docker desktop / orbstack
   https://www.docker.com/ (Docker Desktop) (windows, mac, linux)
   https://orbstack.dev/ (mac)

2. Command to run

```sh
# At the same folder where docker-compose.yaml is located
docker-compose up -d

# At difference folder where docker-compose.yaml is located
docker-compose up -d -f {path to docker-compose.yml}
```
